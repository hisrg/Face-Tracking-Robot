# Face-Tracking-Robot
This project introduces how to build a Face Tracking Robot Using Some commponents of Hardware and Software and algorithm,so it is robot System, There are more detail as below:
I: Material
    Hardware： Surface pro 4 or windows(10) PC, Arduino Mega 2560, Servo driver：Double H bridge /150W/7.5A/12V
    Robot chassis：T500, DC/DC adapter: 12 to 9V /5000mAh/5A double ,Sensors: Speed, encoder，camera，compass
    SDK(IDE): Micorosfot Viusal Studio 2017(community or later), LabVIEW 2016(32bit) or later, Arduino IDE(last),Android studio(last) or TCP/IP dubger assitant for Android Phone.
II:Algorithm
    Speed Control: PID
    Face tracking: CNN http://dataunion.org/11692.html
